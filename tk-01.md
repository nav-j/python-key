import tkinter as tk

# Function to change the label text
def show_message():
    label.config(text="Welcome to Tkinter!")

# Create the main window
window = tk.Tk()
window.title("Simple Tkinter Program")
window.geometry("300x200")

# Create a label
label = tk.Label(window, text="Hello!", font=("Arial", 14))
label.pack(pady=20)

# Create a button
button = tk.Button(window, text="Click Me", command=show_message)
button.pack()

# Run the application
window.mainloop()

# Explaination

Let's understand the code **line by line**.

---

## Step 1: Import the Tkinter library

```python
import tkinter as tk
```

* `tkinter` is Python's built-in library for creating **Graphical User Interfaces (GUI)**.
* `as tk` creates a shorter name (`tk`) so you don't have to write `tkinter` every time.

Example:

```python
tk.Label()
```

instead of

```python
tkinter.Label()
```

---

## Step 2: Create a function

```python
def show_message():
    label.config(text="Welcome to Tkinter!")
```

### Explanation

* `def` is used to define a function.
* The function name is `show_message`.
* This function runs when the button is clicked.
* `label.config()` changes the properties of the label.
* Here it changes the label's text.

Initially:

```
Hello!
```

After clicking the button:

```
Welcome to Tkinter!
```

---

## Step 3: Create the main window

```python
window = tk.Tk()
```

### Explanation

* `Tk()` creates the main application window.
* Everything (buttons, labels, text boxes) will be placed inside this window.

Think of it as the **container** of your application.

---

## Step 4: Give the window a title

```python
window.title("Simple Tkinter Program")
```

This sets the title displayed at the top of the window.

Output:

```
----------------------------
 Simple Tkinter Program
----------------------------
```

---

## Step 5: Set the window size

```python
window.geometry("300x200")
```

This sets the window size.

* Width = **300 pixels**
* Height = **200 pixels**

```
300 pixels
+----------------------+
|                      |
|                      | 200 pixels
|                      |
+----------------------+
```

---

## Step 6: Create a Label

```python
label = tk.Label(window, text="Hello!", font=("Arial", 14))
```

### Explanation

Creates a label inside the window.

Parameters:

* `window` → Parent window.
* `text="Hello!"` → Initial text displayed.
* `font=("Arial", 14)` → Font style and size.

Output:

```
Hello!
```

---

## Step 7: Display the Label

```python
label.pack(pady=20)
```

### Explanation

`pack()` places the widget inside the window.

`pady=20`

* Adds **20 pixels of vertical space** above and below the label.

Without padding:

```
Hello!
Click Me
```

With padding:

```

Hello!

Click Me
```

---

## Step 8: Create a Button

```python
button = tk.Button(window, text="Click Me", command=show_message)
```

### Explanation

Creates a button.

Parameters:

* `window` → Parent window.
* `text="Click Me"` → Text shown on the button.
* `command=show_message` → Calls the `show_message()` function when clicked.

**Notice**

There are **no parentheses** after `show_message`.

Correct:

```python
command=show_message
```

Wrong:

```python
command=show_message()
```

If you write `()`, the function executes immediately when the program starts instead of waiting for a button click.

---

## Step 9: Display the Button

```python
button.pack()
```

Places the button inside the window.

---

## Step 10: Start the GUI

```python
window.mainloop()
```

This is the most important line.

* Starts the Tkinter event loop.
* Keeps the window open.
* Waits for user actions like:

  * Button clicks
  * Keyboard input
  * Mouse movement

Without this line, the window would appear briefly and close immediately.

## Summary of Important Functions

| Code                   | Purpose                                             |
| ---------------------- | --------------------------------------------------- |
| `import tkinter as tk` | Imports the Tkinter library                         |
| `tk.Tk()`              | Creates the main window                             |
| `window.title()`       | Sets the window title                               |
| `window.geometry()`    | Sets the window size                                |
| `tk.Label()`           | Creates a text label                                |
| `tk.Button()`          | Creates a clickable button                          |
| `pack()`               | Places widgets in the window                        |
| `label.config()`       | Changes label properties (such as text)             |
| `command=show_message` | Executes the function when the button is clicked    |
| `window.mainloop()`    | Starts the GUI event loop and keeps the window open |
