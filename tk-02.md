Let's go through this **Tkinter Greeting App** step by step.

---

# Complete Code

```python
import tkinter as tk

def greet():
    name = entry.get()
    result.config(text=f"Hello, {name}!")

window = tk.Tk()
window.title("Greeting App")
window.geometry("300x200")

tk.Label(window, text="Enter Your Name:").pack(pady=5)

entry = tk.Entry(window)
entry.pack()

tk.Button(window, text="Submit", command=greet).pack(pady=10)

result = tk.Label(window, text="")
result.pack()

window.mainloop()
```

---

# Step 1: Import Tkinter

```python
import tkinter as tk
```

### Explanation

* Imports the **Tkinter** library.
* `tk` is an alias (short name) for `tkinter`.

Instead of writing:

```python
tkinter.Label()
```

you can simply write:

```python
tk.Label()
```

---

# Step 2: Create a Function

```python
def greet():
```

### Explanation

This creates a function named `greet()`.

This function will execute **only when the Submit button is clicked**.

---

# Step 3: Get the User Input

```python
name = entry.get()
```

### Explanation

`entry` is the text box where the user types their name.

The `.get()` method retrieves the text entered by the user.

Example:

If the user types:

```
Navjot
```

Then

```python
name = entry.get()
```

stores

```
Navjot
```

inside the variable `name`.

---

# Step 4: Change the Label Text

```python
result.config(text=f"Hello, {name}!")
```

### Explanation

`result` is the output label.

`config()` changes its properties.

Here, it changes the label text.

The `f` before the string indicates an **f-string**, allowing variables to be inserted directly into the text.

Example:

If

```python
name = "Navjot"
```

then

```python
f"Hello, {name}!"
```

becomes

```
Hello, Navjot!
```

---

# Step 5: Create the Main Window

```python
window = tk.Tk()
```

### Explanation

Creates the main application window.

Everything else (labels, buttons, text box) will be placed inside this window.

---

# Step 6: Set the Window Title

```python
window.title("Greeting App")
```

Changes the title displayed at the top of the window.

Output:

```
Greeting App
```

---

# Step 7: Set Window Size

```python
window.geometry("300x200")
```

Sets:

* Width = 300 pixels
* Height = 200 pixels

---

# Step 8: Create a Label

```python
tk.Label(window, text="Enter Your Name:").pack(pady=5)
```

### Explanation

Creates a label that displays:

```
Enter Your Name:
```

Parameters:

* `window` → Parent window
* `text` → Text displayed

`pack(pady=5)` places the label in the window and adds **5 pixels of vertical spacing**.

---

# Step 9: Create an Entry Box

```python
entry = tk.Entry(window)
```

### Explanation

Creates a text box where the user can type input.

Example:

```
+------------------+
| Navjot           |
+------------------+
```

---

# Step 10: Display the Entry Box

```python
entry.pack()
```

Places the text box in the window.

---

# Step 11: Create a Button

```python
tk.Button(window, text="Submit", command=greet).pack(pady=10)
```

### Explanation

Creates a button.

Parameters:

* `window` → Parent window
* `text="Submit"` → Button text
* `command=greet` → Calls the `greet()` function when clicked

**Important:**

Correct:

```python
command=greet
```

Incorrect:

```python
command=greet()
```

Using `()` would execute the function immediately instead of waiting for the button click.

---

# Step 12: Create an Output Label

```python
result = tk.Label(window, text="")
```

### Explanation

Creates an empty label.

Initially, nothing is displayed because the text is an empty string (`""`).

---

# Step 13: Display the Output Label

```python
result.pack()
```

Places the output label in the window.

---

# Step 14: Start the Application

```python
window.mainloop()
```

### Explanation

Starts Tkinter's event loop.

It keeps the window open and listens for user interactions such as:

* Button clicks
* Keyboard input
* Mouse actions

Without `mainloop()`, the window would close immediately.

# Summary of Important Methods

| Code                | Purpose                                                 |
| ------------------- | ------------------------------------------------------- |
| `tk.Tk()`           | Creates the main window                                 |
| `window.title()`    | Sets the title of the window                            |
| `window.geometry()` | Sets the window size                                    |
| `tk.Label()`        | Creates a label to display text                         |
| `tk.Entry()`        | Creates a text box for user input                       |
| `entry.get()`       | Retrieves the text entered in the entry box             |
| `tk.Button()`       | Creates a clickable button                              |
| `command=greet`     | Calls the `greet()` function when the button is clicked |
| `result.config()`   | Updates the text of the result label                    |
| `pack()`            | Places widgets inside the window                        |
| `mainloop()`        | Starts the GUI and keeps the window running             |
